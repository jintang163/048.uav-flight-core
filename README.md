# 无人机飞控系统 (UAV Flight Control System)

一个完整的全自主无人机飞行控制系统，包含飞控固件、地面站、通信链路和多个高级功能模块。

## 技术栈

### 飞控嵌入式
- **硬件**：STM32F4系列
- **操作系统**：FreeRTOS
- **算法**：扩展卡尔曼滤波（姿态解算），串级PID控制
- **传感器**：MPU6050（六轴）、GPS、磁力计、气压计
- **执行器**：PWM输出驱动无刷电机电调
- **协议**：MAVLink v2

### 地面站前端
- **框架**：React 18 + TypeScript
- **UI组件**：Ant Design 5
- **图表**：ECharts 5
- **地图**：高德地图 JS API
- **通信**：WebSocket + MAVLink.js

### 地面站后端
- **语言**：Go 1.21
- **框架**：Gin v1.9
- **ORM**：GORM v1.26
- **数据库**：MySQL 8.0 + Redis 7
- **消息队列**：NSQ
- **存储**：MinIO（日志、航线）
- **监控**：Prometheus + Grafana

### 通信链路
- **传输**：4G/5G + 数传电台
- **协议**：MAVLink v2
- **加密**：TLS 1.3 + 国密SM4对称加密

## MVP 功能

### 1. 飞行控制与姿态解算
- 多传感器数据融合（MPU6050 + GPS + 磁力计 + 气压计）
- 扩展卡尔曼滤波姿态解算（四元数→欧拉角）
- 串级PID控制器（角度环 + 角速度环）
- 支持SBUS/PPM遥控器和地面站指令
- 姿态解算频率：500Hz，控制频率：200Hz
- 悬停精度：水平±0.5m，垂直±0.3m

### 2. 航线规划与自主飞行
- 地图可视化航点编辑
- 航点动作配置（悬停、拍照、盘旋、降落）
- 航线模板保存与复用
- 断点续飞功能
- 低电量自动返航
- 信号丢失自动返航

### 3. 地面站实时监控
- 姿态地平仪、高度、空速仪表盘
- 电压、电流、GPS星数显示
- 地图实时轨迹绘制
- 一键起飞/降落/返航/解锁
- 异常语音告警及短信通知

## 高级功能模块

- 电子围栏（Geofence）
- 多机编队（Formation Flight）
- 任务载荷（Payload Control）
- 黑匣子日志（Blackbox）
- 智能电池管理（Smart Battery）
- 避障系统（Obstacle Avoidance）
- 远程驾驶舱（Remote Cockpit）
- 气象预警（Weather Alert）
- RTK精准降落（RTK Landing）
- 固件远程升级（OTA）

## 快速开始

```bash
# 启动基础设施
docker-compose up -d mysql redis minio nsq prometheus

# 启动后端
cd groundstation-backend
go mod download
go run cmd/server/main.go

# 启动前端
cd groundstation-frontend
npm install
npm start
```
